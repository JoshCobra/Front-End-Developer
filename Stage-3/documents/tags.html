Tags In HTML

The importance of Tags

We use tags in text to highlight words in our text

For example, we can use the <b> tag to make text bold.

We can use the <i> tag to make text italic.

We can use the <u> tag to underline text.

We can use the <mark> tag to highlight text.

We can use the <del> tag to strikethrough text.

We can use the <em> tag to emphasize text.

You can use tags in all cases

  • bold text
  • italic text
  • underlined text


  • highlighted text
  • strikethrough text
  • emphasized text
